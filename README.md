# 🪪 Personal Business Card App

A fully programmatic UIKit iOS app that displays a personal business card 
with interactive contact options and editable details.

## 📱 Features

- View personal details — name, designation, company, photo
- Tap Phone to view number, copy to clipboard or call directly
- Tap Email to view address, copy to clipboard or send mail
- One tap LinkedIn and GitHub profile opening in Safari
- Edit all card details with persistent storage via UserDefaults
- Data survives app restarts

## 🛠️ Built With

- Swift
- UIKit (100% programmatic — no storyboards)
- UserDefaults for persistence
- Auto Layout with NSLayoutConstraint
- UIStackView for layout
- UIButton.Configuration with SF Symbols
- URL schemes — tel://, mailto:, https://
